2 邮件配置








2.1 接收邮件(offlineimap)





offlineimap 是一个python程序, 用来同步IMAP服务器的邮件, 不支持POP3协议.





通过offlineimap将邮件下载到本地, 再通过mu/mu4e进行管理








2.1.1 安装





debian/ubuntu 通过apt安装





$ sudo apt install offlineimap












2.1.2 配置





offlineimap默认的配置文件在 ~/.offlineimaprc , 首先创建 .offlineimaprc 文件.





$ touch ~/.offlinemaprc








[general]
accounts = qssec
fsync = False

[Account 163]
localrepository = 163-Local
remoterepository = 163-Remote
# 配置为True 以支持中文文件夹
utf8foldernames = True

[Repository qssec-Local]
type = Maildir
# 邮件保存的路径
localfolders = ~/Workspace/mails/
nametrans = lambda foldername: foldername.decode('imap4-utf-7').encode('utf-8')

[Repository qssec-Remote]
sslcacertfile=/etc/ssl/certs/ca-certificates.crt
maxconnections = 2
type = IMAP
auth = on
ssl = yes
remotehost = imap.163.com

# 填写邮箱地址
remoteuser = YourEmailXXX@163.com
remoteport = 993

#填写密码，不是登录密码，而是授权码, 通过登陆邮箱主页,生成授权码
remotepass = XXXXXXXXXXXXXXXXXXXXX

nametrans = lambda folder: folder.decode('imap4-utf-7').encode('utf-8')

folderfilter = lambda folder: folder in ['INBOX']
keepalive = 60
holdconnectionopen = yes












2.1.3 同步邮件






$ offlineimap














2.2 mu/mu4e





mu4e是基于emacs的邮件客户端, 基于mu进行索引和搜索.








2.2.1 安装






$ sudo apt install mu4e













2.2.2 配置






;; 根据您自己的 mu4e 安装路径替换 load-path, debian 10 为以下路径
(add-to-list 'load-path "/usr/share/emacs/site-lisp/mu4e")

(require 'mu4e)

;; 配置 mu4e 作为 MUA 以及基本的邮件信息配置
(setq mail-user-agent 'mu4e-user-agent)
(setq user-full-name "YourName")
(setq user-mail-address "YourEmailXXX@163.com")

;; 配置使用 offlieimap 收取邮件
(setq mu4e-get-mail-command "offlineimap"   ;; 使用 offlineimap 获取邮件
      mu4e-update-interval 120)             ;; 每两分钟检查邮件并更新索引 (配置单位：秒)

;; 配置默认 smtp 服务以使用 smtpmail mode 发送 Email
;; SMTP 服务需要认证，我采用 ~/.authinfo.gpg 的方式提供认证信息。
;; 有关 SMTP 认证, 参阅 https://www.emacswiki.org/emacs/SmtpAuth
(setq send-mail-function 'smtpmail-send-it
      message-send-mail-function 'smtpmail-send-it
      smtpmail-smtp-server  "smtp.163.com"
      smtpmail-smtp-service "465"
      ;; smtpmail-auth-credentials (())
      smtpmail-auth-credentials "~/.authinfo.gpg"
      smtpmail-stream-type  'starttls)

(setq mu4e-maildir "~/Workspace/mails"
      ;;mu4e-trash-folder "/Trash"
      ;;mu4e-refile-folder "/Archive"
      ;;mu4e-sent-folder "/Sent"
      ;;mu4e-drafts-folder "/Drafts"
	      mu4e-compose-signature-auto-include nil
      mu4e-view-show-images t
      mu4e-view-show-addresses t)

;; 配置桌面通知
(with-eval-after-load 'mu4e-alert
  ;; Enable Desktop notifications
  (mu4e-alert-set-default-style 'notifier)) ; For linux

;; Bookmarks 允许结合 b (先导快捷键) 自定义快捷键以快速搜索 Email
;; configure the bookmarks.
(setq mu4e-bookmarks
      '( ("flag:unread AND NOT flag:trashed"                    "Unread messages"                  ?u)
	 ("date:today..now"                                     "Today's messages"                 ?t)
	 ("date:7d..now"                                        "Last 7 days"                      ?w)
	 ("date:1d..now AND NOT list:emacs-orgmode.gnu.org"     "Last 1 days"                      ?o)
	 ("date:1d..now AND list:emacs-orgmode.gnu.org"         "Last 1 days (org mode)"           ?m)
	 ("maildir:/sent"                                       "sent"                             ?s)
	 ("maildir:/drafts"                                     "drafts"                           ?d)
	 ("mime:image/*"                                        "Messages with images"             ?p)))

;; Show mu4e maildirs summary in mu4e-main-view
(use-package mu4e-maildirs-extension
  :init
  (mu4e-maildirs-extension)
  )

(use-package mu4e-views)
(use-package mu4e-overview)

(provide 'init-mu4e)

3 emacs画图





语言设置
设置 org-babel-load-languages 变量 增加允许语言设置, ditaa 为 t





(org-babel-do-load-languages
 'org-babel-load-languages
 '(;; other Babel languages
   (ditaa . t)
   (plantuml . t)
   (dot . t)
   (xxx . t)
   ))
;;生成图像时不予提示

(setq org-confirm-babel-evaluate nil)

(setq org-ditta-jar-path (expand-file-name "~/.emacs.d/elpa/contrib/scripts/ditta.jar"))
(setq org-plantuml-jar-path (expand-file-name "~/.emacs.d/elpa/contrib/scripts/plantuml.jar"))







这样设置后,可以按照以下方式调用:





#+BEGIN_SRC ditaa :file some_filename.png :cmdline -r -s 0.8
#+BEGIN_SRC plantuml :file export-file-name :cmdline -charset UTF-8
如果不进行以上设置,通过begin_src将不会成功,需要按照以下方式调用:





#+begin_ditaa test.png -r -s 0.8
