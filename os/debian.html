2.3. vncserver

tigervncserver 

$ man tigervncserver

工作目录： $HOME/.vnc

配置文件： $HOME/.vnc/vnc.conf /etc/vnc.conf

桌面环境启动脚本： $HOME/.vnc/Xvnc-session

password文件： $HOME/.vnc/passwd

Xvnc-session文件内容 

  $ cat Xvnc-session
#! /bin/sh

test x"$SHELL" = x"" && SHELL=/bin/bash
test x"$1"     = x"" && set -- default

vncconfig -iconic &
i3 &

vncserver@.service内容： 

# Vncserver service file for Debian or Ubuntu with systemd
#
#  Install vncserver and tools
#  e.g. apt-get install tightvncserver autocutsel gksu
#
# 1. Copy this file to /etc/systemd/system/vncserver@:1.service
# 2. Edit User=
#    e.g "User=paul"
# 3. Edit the vncserver parameters appropriately in the ExecStart= line!
#    e.g. the -localhost option only allows connections from localhost (or via ssh tunnels)
# 4. Run `systemctl daemon-reload`
# 5. Run `systemctl enable vncserver@:<display>.service`
#

[Unit]
Description=Remote desktop service (VNC)
After=syslog.target network.target

[Service]
Type=forking
User=fanyf
WorkingDirectory=/home/fanyf

# Clean any existing files in /tmp/.X11-unix environment
ExecStartPre=/bin/sh -c '/usr/bin/vncserver -kill %i > /dev/null 2>&1 || :'
#ExecStart=/usr/bin/vncserver -geometry 1800x1000 -depth 16 -dpi 120 -alwaysshared -localhost %i
ExecStart=/usr/bin/vncserver %i
ExecStop=/usr/bin/vncserver -kill %i

[Install]
WantedBy=multi-user.target

启动vncserver 

$ sudo cp vncserver@.service /etc/systemd/system/
$ sudo systemctl enable vncserver@:1
$ sudo systemctl start vncserver@:1

2.4. linux电源相关的ACPI事件设置

systemd-logind处理相关的事件

配置文件为: 

/etc/systemd/logind.conf
/etc/systemd/logind.conf.d/*.conf
/run/systemd/logind.conf.d/*.conf
/usr/lib/systemd/logind.conf.d/*.conf

文件说明 

####### 可以执行的动作 #############
# ignore  无操作
# poweroff 关机
# reboot  重启
# halt 关机，但不关闭电源
# kexec
# suspend 睡眠
# hibernate 休眠
# hibrid-sleep 同时休眠到内存和硬盘
# suspend-then-hibernate 先睡眠，超时后休眠
# lock 锁屏
####################
#电源键的行为
#HandlePowerKey=poweroff
#挂起键的行为
#HandleSuspendKey=suspend
#休眠键的行为
#HandleHibernateKey=hibernate

#HandleLidSwitch=suspend
#如果使用外部电源，且此变量不是默认值，则合上盖子使用此值
#HandleLidSwitchExternalPower=suspend
#如果主机插入了扩展坞或者连接多个显示器时，合上盖子的行为
#HandleLidSwitchDocked=ignore

2.5. qv2ray

NOTE: Replace $YOUR_DISTRIBUTION with your distribution code (stable/unstable/focal/groovy…) below. 

# Install some prerequisites needed by adding GPG public keys
$ sudo apt-get install gnupg ca-certificates curl

# Import our GPG key. Notice the hyphen at the end of line.
$ curl -sSL https://qv2ray.net/debian/pubkey.gpg | sudo apt-key add -

# Add the our official APT repository:
$ echo "deb [arch=amd64] https://qv2ray.net/debian/ $YOUR_DISTRIBUTION main" | sudo tee /etc/apt/sources.list.d/qv2ray.list

# To update the APT index:
$ sudo apt-get update

# You can install Qv2ray from APT now:
$ sudo apt-get install qv2ray
