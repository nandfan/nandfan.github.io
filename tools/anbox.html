1 安装

1.1 安装anbox

$ sudo apt install anbox

$ ls -1 /dev/{ashmem,binder}

1.2 依赖内核模块

$ ls -1 /dev/{ashmem,binder}

# 如果不存在

$ sudo modprobe ashmem_linux
$ sudo modprobe binder_linux

1.3 Android image

Anbox 会使用 /var/lib/anbox/android.img. 作为android镜像。

下载地址： https://build.anbox.io/android-images/ 

$ sudo mv ~/Downloads/android_amd64.img /var/lib/anbox/android.img

重启服务： 

$ sudo service anbox-container-manager restart

从桌面或者命令行启动： 

anbox launch --package=org.anbox.appmgr --component=org.anbox.appmgr.AppViewActivity

Anbox in XFCE Whisker Menu

1.4 安装adb:

$ sudo apt install adb

APK下载地址：https://www.apkmirror.com/

安装apk： 

$ adb install XXX.apk
