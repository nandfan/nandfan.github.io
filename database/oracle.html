切换实例归档

select group_number, name, total_mb, free_mb, total_mb-free_mb used_mb from v$asm_disk_stat;
select group_number, name, total_mb, free_mb, total_mb-free_mb used_mb from v$asm_diskgroup;
查询asm磁盘状态
select GROUP_NUMBER,DISK_NUMBER,MOUNT_STATUS,HEADER_STATUS,MODE_STATUS,name,path from v$asm_disk;
查询表空间使用率
select total.tablespace_name,
       round(total.MB, 2) as Total_MB,
       round(total.MB - free.MB, 2) as Used_MB,
       round((1 - free.MB / total.MB) * 100, 2) || '%' as Used_Pct
       from (select tablespace_name, sum(bytes) / 1024 / 1024 as MB
                    from dba_free_space
                    group by tablespace_name) free,
       (select tablespace_name, sum(bytes) / 1024 / 1024 as MB
               from dba_data_files
               group by tablespace_name) total
       where free.tablespace_name = total.tablespace_name;

2. RAC集群

2.1. rac集群启动和关闭顺序

实例名称为 qssecdb 四个节点分别为 node-rac1 node-rac2 node-rac3 node-rac4

以oracle 用户执行: 停止所有节点上的实例 

$ srvctl stop database -d qssecdb

每个节点执行: 

$ emctl stop dbconsole

关闭所有asm实例 

$ srvctl stop asm -n node-rac1 -i +ASM1
$ srvctl stop asm -n node-rac2 -i +ASM2
$ srvctl stop asm -n node-rac3 -i +ASM3
$ srvctl stop asm -n node-rac4 -i +ASM4

停止所有应用: 

$ srvctl stop nodeapps -n node-rac1
$ srvctl stop nodeapps -n node-rac2
$ srvctl stop nodeapps -n node-rac3
$ srvctl stop nodeapps -n node-rac4

以root用户执行: 

# crsctl stop cluster -all   //停止所有节点服务

或者每个节点执行: 

# crsctl stop cluster

或者 执行 

# crsctl stop cluster -n qssecdb1 qssecdb2 qssecdb3 qssecdb4

3. ASM存储

3.1. 确定asm存储设备

配置确定方法

  1. 确定盘符挂载路径 

    # /usr/sbin/oracleasm status
# /usr/sbin/oracleasm listdisks            #查看磁盘路径

  2. 查看归档日志路径 

    > sqlplus / as sysdba
> show parameter db_recovery   #确定路径名称  ARCH

  1. 查看归档路径对应的磁盘名称 

    > select a.name from V_$ASM_DISK a, V_$ASM_DISKGROUP b where a.group_number = b.group_number and b.name = 'ARCH';

  2. 确定磁盘路径是否存在 

    find /dev/oracleasm/ -name ARCHDISK;

> select a.group_number, a.disk_number, a.path from V_$ASM_DISK a, V_$ASM_DISKGROUP b where a.group_number = b.group_number  and b.name = 'DATA';
