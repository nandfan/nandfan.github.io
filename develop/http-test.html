1 Httperf和Autobench

1.1 介绍

Httperf是hp使用的一款性能测试工具，可以测试WEB服务器性能，支持HTTP/1.0、HTTP/1.1和HTTPS，使用比较简单；

Autobench 是一款基于httperf的Perl脚本，实现了自动化过程，它可以在一次测试中调用多次httperf来进行测试；

2 wrk

项目地址： git@github.com:wg/wrk.git

2.1 编译

$ git clone git@github.com:wg/wrk.git
$ cd wrk
$ make

2.2 选项

-c, --connections: 保持打开的http连接数，每个线程有connections/threads个连接
-d, --duration:    持续测试时间, e.g. 2s, 2m, 2h
-t, --threads:     使用的线程数量
-s, --script:      luajit脚本, 详情查看 SCRIPTING 文件
-H, --header:      自定义添加请求header e.g. "User-Agent: wrk"
    --latency:     打印详细的延迟统计信息
    --timeout:     设置相应的超时时间
